# 🏙️ Street-Gallery | E-commerce Platform

[![Python](https://img.shields.io/badge/Python-3.10+-3776AB?style=for-the-badge&logo=python&logoColor=white)](https://www.python.org/)
[![Django](https://img.shields.io/badge/Django-5.2+-092E20?style=for-the-badge&logo=django&logoColor=white)](https://www.djangoproject.com/)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg?style=for-the-badge)](https://opensource.org/licenses/MIT)

**Street-Gallery** es una plataforma de comercio electrónico de alto rendimiento diseñada para la venta de indumentaria urbana y arte curado. Construida con **Django**, la aplicación ofrece una experiencia de usuario fluida, desde la navegación de productos hasta el procesamiento de pagos seguro.

---

## 🚀 Características Principales (Core Features)

* **Gestión de Catálogo:** Sistema dinámico de productos con categorías y detalles técnicos.
* **Carrito de Compras Pro:** Persistencia de productos y actualización de cantidades en tiempo real.
* **Autenticación de Usuarios:** Sistema completo de registro, login y gestión de perfiles para clientes.
* **Integración con PayPal:** Procesamiento de pagos seguro y automatizado.
* **Notificaciones:** Confirmación de pedidos mediante envío de correos electrónicos automáticos.
* **Diseño Responsive:** Optimizado para dispositivos móviles y escritorio.

---

## Tecnologías Utilizadas (Built With)

* **Backend:** [Django](https://www.djangoproject.com/) (Python Framework)
* **Base de Datos:** SQLite (Desarrollo) / PostgreSQL (Producción)
* **Pagos:** PayPal Checkout SDK
* **Frontend:** HTML5, CSS3 (Less), JavaScript
* **Estilos:** Bootstrap 5 & Custom Techwear Design
